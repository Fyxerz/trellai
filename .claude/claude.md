# test2

## Design Decisions

This project uses Next.js, React, Tailwind CSS, Drizzle ORM, Socket.IO, TypeScript, Vitest as its core technology stack.

## Coding Principles

- **TDD is primordial** — Write tests before implementation. Every feature should start with a failing test.
- Write clean, readable code with meaningful names and clear intent.
- Prefer small, focused functions and modules.
- Keep dependencies minimal and well-justified.
